Download Link and Button

You can provide a downloadable resource by using DownloadButton or DownloadLink

DownloadButton#

import {DownloadButton} from '@theme/Download';
<DownloadButton path="/datasets/vgsales.csv" />

DownloadLink#

import {DownloadLink} from '@theme/Download';
The DownloadLink <DownloadLink path="/datasets/vgsales.csv" text="does not"/> break the text flow.

The DownloadLink does not break the text flow.

danger

The path root is in /static and has to start with a backslash. datasets/vgsales.csv or ./datasets/vgslaes.csv will not be accepted!

Optional Arguments#

Optional Arguments are

  • text : the text inside the link or button
  • name : the name of the downloaded file
import {DownloadButton} from '@theme/Download';
<DownloadButton path="/datasets/vgsales.csv" text="example dataset" name="dataset.csv"/>
info

if you provide a name you have to include the file extension
