## Getting Started
### What's Streamlit?
Streamlit is an open-source Python library that makes it easy to create and share beautiful, custom web apps for machine learning and data science.
[source_streamlit](https://docs.streamlit.io/).

You may find many open-source apps wriiten in streamlit such as e.g. the following NLP apps: [a demo of an spacy NLP pipeline](https://share.streamlit.io/ines/spacy-streamlit-demo/app.py) or [an analysis of the game of thrones characters](https://got-streamlit.herokuapp.com/).
